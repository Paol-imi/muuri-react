Muuri-react

The layout engine for React

Get Started

Easy to Use

Muuri-react has been designed to work out of the box with most of its features, while still guaranteeing a vast possibility of configurations.

Fully customizable

At it's very core Muuri is a layout engine which is limited only by your imagination. You can build any kind of layout, asynchronously in web workers if you wish.

Focus on What Matters

The long-term goal of Muuri is to provide a simple API for building amazing layouts with unmatched performance and most of the complexity abstracted away.
